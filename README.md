## :inbox_tray: Install

### `conanfile.txt`
Add the following line to your `conanfile.txt` file under the `[requires]`
section:

```
libhal/[x]
```

Replace `[x]` with the appropriate version number.

### `conanfile.py`

Add the following to your `def requirements(self):` function.

```python
self.requires("libhal/[x]")
```

Replace `[x]` with the appropriate version number.
